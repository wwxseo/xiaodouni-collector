## 🛠️ 功能特性
- **全自动更新**：利用 GitHub Actions，每天定时抓取。
- **无需 API**：直接使用网页解析技术，简单高效。
- **自适应排版**：采用 HTML 布局，支持横向排列和自动换行。
- **零成本运行**：完全利用 GitHub 提供的免费服务器资源。

## 🚀 快速上手
如果你也想拥有一个自己的小豆泥仓库：
1. **Fork** 本项目。
2. 进入你 Fork 后的仓库，点击 **Actions**，点击 `Xiaodouni Daily Update` 并启用它 (Enable workflow)。
3. 手动点击 **Run workflow** 即可看到效果！
